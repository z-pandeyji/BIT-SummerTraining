# Artificial Intelligence Class Materials

This folder contains local class notes and notebooks for the Artificial Intelligence module.

The folder currently keeps the existing local spelling `Artifical_intelligence` so older local links and paths continue to work.

## Batch A

- `Day1.md` through `Day5.md`: AI class notes and daily topic details.
- `Vector_Basic.ipynb`, `vector_DB.ipynb`: vector and vector database practice.
- `Embeddings_LAB.ipynb`, `Embeddings_LAB (1).ipynb`: embedding experiments.
- `Token_Budget_Planner.ipynb`: token budget planning practice.
- `Baby_GPT (1).ipynb`: classroom language-model experiment.
- `AI_project_1.ipynb`: AI project notebook.

## Batch B

- `Day1.md` through `Day4.md`: AI class notes and daily topic details.
- `Token_Cost_Context_Calculator.ipynb`: token, cost, and context-window practice.
- `HF_transformer.ipynb`: Hugging Face transformer practice.
- `Baby_GPT.ipynb`: classroom language-model experiment.

## Notes

- Do not commit API keys, private prompts, student private data, or paid model credentials.
- Large model files and generated cache folders should stay outside the repository.
- Run notebooks locally and commit only the notebooks or small supporting files needed for class review.
