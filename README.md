# Python, Data Analytics, Data Science, Machine Learning & AI Assignments

This repository is for class assignments, practice work, and learning resources.

Students will submit every assignment through a Pull Request. This helps everyone learn Git, GitHub, clean code, and review-based development.

## Repository Structure

```txt
assignments/
```

## Student Rules

1. Create your own branch for every assignment.
2. Do not edit another student's folder.
3. Submit assignments through Pull Request only.
4. Use your real name for your folder.
5. Keep your code clean, readable, and properly named.
6. Put your work inside your batch folder only.

## Submission Format

Use this format:

```txt
assignments/day-01/submissions/batch-a/your-name/solution.py
assignments/day-01/submissions/batch-b/your-name/solution.py
```

Example:

```txt
assignments/day-01/submissions/batch-a/rahul/solution.py
assignments/day-01/submissions/batch-b/priya/solution.py
```

## Student Flow

Clone the repository:

```bash
git clone <repo-link>
cd BIT-SummerTraining
```

Create your assignment branch:

```bash
git checkout -b day-01-your-name
```

Create your solution file:

```txt
assignments/day-01/submissions/batch-a/your-name/solution.py
```

Commit and push:

```bash
git add .
git commit -m "Add Day 1 assignment by Your Name"
git push origin day-01-your-name
```

Then open GitHub and create a Pull Request.
